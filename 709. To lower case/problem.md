## Problem Statement
Given a string `s`, return the string after replacing every uppercase letter with the same lowercase letter.

### Example 1:
**Input:**
```python
s = "Hello"
```
**Output:**
```python
"hello"
```

### Example 2:
**Input:**
```python
s = "here"
```
**Output:**
```python
"here"
```

### Example 3:
**Input:**
```python
s = "LOVELY"
```
**Output:**
```python
"lovely"
```

## Constraints
- `1 <= s.length <= 100`
- `s` consists of printable ASCII characters.
